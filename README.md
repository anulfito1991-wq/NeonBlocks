# NeonBlocks — Web

Static site for the NeonBlocks iOS game. Currently hosts the App Store-required privacy policy.

## What's here

- [`index.html`](./index.html) — Privacy Policy, self-contained (no CDN deps, no trackers)

## URL

Served via GitHub Pages at https://anulfito1991-wq.github.io/NeonBlocks/

## Editing

The HTML is deliberately single-file and dependency-free. Edit in place, commit, push — GitHub Pages redeploys in ~60 seconds.

If you later add a support page, marketing landing, or changelog, drop them in as additional `.html` files at the repo root. The URL pattern will be `https://anulfito1991-wq.github.io/NeonBlocks/<filename>.html`.
